## The Positions Explained

**Platonism**: Numbers exist independently of us, like mountains we discover rather than build. The challenge: how do physical brains access non-physical abstract objects?

**Fictionalism**: Math is useful fiction, like Sherlock Holmes. Literally false, but enormously helpful. The challenge: why is this particular fiction so unreasonably effective?

**Formalism**: Math is a game with symbols and rules. Don't ask what numbers "really are" ‚Äî just play the game. The challenge: G√∂del showed formal systems can't capture all mathematical truth.

**Intuitionism**: Math exists only when we construct it through mental acts. No completed infinities, no proofs by contradiction. The challenge: rejects much of classical mathematics.

**Structuralism**: Numbers aren't objects ‚Äî they're positions in patterns. "2" is whatever occupies the 2-position in a number-like structure. The challenge: aren't structures themselves abstract objects?
